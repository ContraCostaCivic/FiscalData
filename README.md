# FiscalData
Data About the Financial Status of Contra Costa County Local Governments

This repo contains fiscal data and analysis for local governments in Contra Costa County.

Each city and town in the County produces a Comprehensive Annual Financial Report.  Many have published reports for the Fiscal Year Ended June 30, 2017. For others, the latest is June 30, 2016.  You can see a repository of California CAFRs here:  http://www.govwiki.info/pdfs/General%20Purpose/?state=CA&year=

You can find budgets and other financial reports on each community's web site.
